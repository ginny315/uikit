AgentSys 是一个开源的 LLM Agent 管理平台，帮助你定义、部署、编排和监控 AI Agent。 像管理微服务一样管理你的 Agent 集群 — 从单机到千级规模，尽在掌控。

快速开始 查看文档 GitHub
agentsys-cli — zsh
$ agentsys agent create
name: code-reviewer
model: claude-opus-4-8
✓ Agent code-reviewer created
$ agentsys task submit --agent code-reviewer
✓ Task task_7a3f queued (priority: high)
$ agentsys workflow run --dag ci-pipeline
▸ lint (2.1s) ▸ test (8.4s) ▸ review (12.7s)
✓ Workflow ci-pipeline completed
$
核心能力

八大功能模块，一站式 Agent 管理

从 Agent 生命周期管理到成本控制，从任务调度到工作流编排 — AgentSys 覆盖了 LLM Agent 生产化的全部环节。

Agent 管理

Agent 全生命周期管理 — 创建、配置、启动、停止、版本回滚，状态机驱动的生命周期保障。

CRUD 版本管理 能力标签

任务调度

优先级队列、公平调度、指数退避重试、死信处理 — 确保每个任务都被可靠执行。

优先级队列 重试策略 死信队列

监控大盘

实时 Agent 状态、任务吞吐、Token 消耗、P50/P95/P99 延迟 — Prometheus + Grafana 集成。

实时状态 P95 延迟 Token 统计

日志与追踪

结构化任务执行日志、完整执行时间线、全文检索 — 基于 ClickHouse 的高性能日志存储。

执行日志 时间线 全文检索

工作流编排

可视化 DAG 编辑器，支持并行/串行步骤、条件分支 — 用拓扑排序自动解析执行顺序。

DAG 编辑器 并行执行 条件分支

成本管理

按 Agent/用户/时间维度的成本报表、Token 配额管控、费用预警 — 让每一次调用都有据可查。

成本报表 配额管控 费用预警

访问控制

RBAC 角色权限、API Key 管理、审计日志 — 满足企业与个人开发者不同粒度的安全需求。

RBAC API Key 审计日志

Webhook 与集成

事件驱动的 Webhook 推送、Slack / GitHub / 企业微信等第三方集成 — 无缝融入你的工具链。

Webhook Slack GitHub
